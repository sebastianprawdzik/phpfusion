Description

Event Management System is an infusion which can be used to organise Events in website. It also provides a nice professional Calendar for ease of browsing events.

Features

  • Descriptive Infusion Admin Panel for event related settings and display settings for Calendar.
  • Member can post events(optional)
  • Comments and Ratings on Events(optional)
  • Ease in Posting event with an event image
  • Events visibility on basis of Guest/Members/User groups
  • Events can be sorted on basis on Title/Author/Schedule
  • Calendar page to browse Events by Day or Month
  • Nice and professional Side Calendar panel
  • Display Birthdays in Calendar(optional)
  • Settings in Admin panel to change colors for calendar

Install

Upload all the files in the /files folder to your PHP-Fusion site root.
Go to your site Admin >>> System >>> Infusions and infuse Event Management System.
Done! A Link to Events page will be automatically added in the Site Links.
To enable side Calendar Panel, go to Admin >>> System >>> Panels and enable the Event Management System Panel.
